# vcpkg_replace_string

Replace a string in a file.

## Usage

```cmake
vcpkg_replace_string(<filename> <match> <replace>)
```

## Parameters

### `<filename>`
The path to the file to modify.

### `<match>`

The string to match.

### `<replace>`

The string to replace all matched sequences with.
